Audio codec ids

val get_name : id -> string

Return the name of the codec.

val find_id : string -> id

Return the id of a codec from his name.

  • Raises Failure if the codec is not found or is not an audio codec.
val find_best_sample_format : id -> Sample_format.t

Return the best sample format of a codec.

  • Raises Failure if the codec has no best sample format.
val get_id : audio Avcodec.t -> id

Return the id of the codec.

val get_channel_layout : audio Avcodec.t -> Channel_layout.t

Return the channel layout set for the codec.

val get_nb_channels : audio Avcodec.t -> int

Returns the number of channels set for the codec.

val get_sample_format : audio Avcodec.t -> Sample_format.t

Returns the sample format set for the codec.

val get_bit_rate : audio Avcodec.t -> int

Returns the bit rate set for the codec.

val get_sample_rate : audio Avcodec.t -> int

Returns the sample rate set for the codec.
